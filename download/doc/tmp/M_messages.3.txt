NAME

M_messages(3fp) - [M_messages]module collecting various general routines for displaying messages

DESCRIPTION

  * LINEART

    junbad
        print eye-catching ASCII graphic (skull)
    junbat
        print eye-catching ASCII graphic (bat)
    junbuster
        print eye-catching ASCII graphic (ghostbuster)
    jundragon
        print eye-catching ASCII graphic (dragon) with message
    junroach
        print eye-catching ASCII graphic (roach)
    junsun
        print eye-catching ASCII graphic (sunshine)
    juntrolls
        print eye-catching ASCII graphic (trolls) with message

  * ILLUMINATE

    tabgraph
        write columns of numbers with a text scale to the right

  * BLOCK LETTERS

    blocks
        write out 132-character string in large block letters
    signs
        write up to 132 large block letters

  * PROGRESS AND STATUS

    percent_done
        place a non-advancing status counter on terminal display
