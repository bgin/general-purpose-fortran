NAME

    _logname(1f) - [FUNIX]display login name

CONTENTS

    Synopsis
    Description
    Options
    Example

SYNOPSIS

    _logname [-help|-version]

DESCRIPTION

    Demonstrate call to system_getlogin(3f) (which calls getlogin(3c)

OPTIONS

        --help display command help and exit

        --version output version information and exit

EXAMPLE

    Command usage:

            _logname
