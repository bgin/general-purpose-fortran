NAME

cprint(1f) - [FILE FILTER] filter prints specified columns

SYNOPSIS

cprint [columns ][-delimiters delim] |-help|-version

DESCRIPTION

cprint is a filter that prints the specified columns

OPTIONS

columns numbers indicating which columns to print -delimiters input column delimiter character(s) (default: whitespace) -help display command help and exit -version display command metadata and exit
